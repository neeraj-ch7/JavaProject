# ❌⭕ Tic Tac Toe (Java)

## 📌 Overview

This is a simple **console-based Tic Tac Toe game** built using Java.
Two players can play the game by entering row and column values.

---

## 🎮 Features

* 2 Player mode (X and O)
* Console-based interface
* Input validation
* Win and draw detection

---

## 🛠️ Tech Stack

* Java
* Scanner (for user input)

---

## 🚀 How to Run

```bash
# Compile the program
javac TicTacToe.java

# Run the program
java TicTacToe
```

---

## 🎯 How to Play

* The board is a 3x3 grid
* Players take turns entering:

  * Row (0–2)
  * Column (0–2)
* First player uses **X**
* Second player uses **O**
* First to align 3 marks (row, column, diagonal) wins
